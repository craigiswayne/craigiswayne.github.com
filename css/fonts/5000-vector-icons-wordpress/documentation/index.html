Installation
Follow the following steps to install the plugin (WordPress version 3.3.x, 3.4.x, 3.5.x or 3.6).
Step 1. Method 1: Unzip the downloaded zip file.
Find and Upload the "cc_icons.zip" file. To upload go to your WordPress admin panel > Plugins > Add New > Upload this zip file > Click on Install Now. This will install the plugin. After installation click on "Activate" link to activate the plugin.
You will find a new plugin "5000+ Vector Icons - WordPress" in the activated plugins list.
You can also follow the other method.
Method 2: Unzip the "cc_icons.zip" file. Copy the "cc_icons" folder in the plugins directory of your WordPress installation.
(Plugin directory of default WordPress installation is your_rooth_path/wp-content/plugins/).
Go to your WordPress Admin panel > Plugins. You will find a new plugin "5000+ Vector Icons - WordPress" in the list. Now click on "Activate" link below it.


Step 2. Now in your WordPress Admin panel > Settings menu you will see a new link added named as "5,000+ Vector Icons". Click on it to go to "5,000+ Vector Icons" Panel.
Note: This plugin activation will create 1 new table in your WordPress database named as YOUR-TABLE-PREFIX_iconswpbyjboncc_vectoricons.


Step 3. If the installation is done correctly, then you will see a list on Icon Sets in 5,000+ Vector Icons Panel on this page. Click on any icon set to get started.
If you can see the icons loaded, then congratulations you have installed the plugin correctly.
Note: If you get some warning / notice / error while loading icons related to file_get_contents, Then please try doing as instructed below:

Please find and change allow_url_fopen = Off  to  allow_url_fopen = On in your php.ini file. Please uncomment it (by removing # from start of this line) if it is already set to allow_url_fopen = On. You can also ask your server administrator to do the same for you.


Uninstall
Follow the following steps to uninstall the plugin (WordPress version 3.3.x, 3.4.x, 3.5.x or 3.6).
Step 1. Go to your WordPress Admin panel > Plugins. You will find a plugin 5,000+ Vector Icons - WordPress in the list. Now click on "Deactivate" link below it.

This will uninstall the plugin.
Note: Uninstallation of the plugin will delete all the icons and associated database tables permanently.









Code Implementation
Below is the example and a breif of a Shortcode generated.
A shortcode is a simple set of functions for creating macro codes for use in post content. A shorted generated to insert a icon looks like as below:

[i font="v1" code="e04c" size="36px" onsize="44px" color="#ffffff" oncolor="#ffffff" padding="16px" onpadding="12px" bgtype="gradient" bg1="#17d15b" bg2="#12b542" onbg1="#ef4f37" onbg2="#d32721" borderwidth = "1px" bordercolor = "transparent" borderstyle = "solid" borderradius = "50%" effect = "spin" link = "http://www.example.com" target = "self" align = "left" margin = "5px 8px 2px 4px"]

This shortcode can you inserted inside a page or post in editor to display icons. Also this shortcode can you used to display icons in any area of the website other than post or page content too. Use do_shortcode(MY_SHORTCODE_HERE); as shown below:

<?php do_shortcode([i font="v1" code="e04c" size="36px" onsize="44px" color="#ffffff" oncolor="#ffffff" padding="16px" onpadding="12px" bgtype="gradient" bg1="#17d15b" bg2="#12b542" onbg1="#ef4f37" onbg2="#d32721" borderwidth = "1px" bordercolor = "transparent" borderstyle = "solid" borderradius = "50%" effect = "spin" link = "http://www.example.com" target = "self" align = "left" margin = "5px 8px 2px 4px"]); ?>


Plugin also helps in generating the HTML code along with the shortcode. You can use this HTML code too to display icons anywhere on your website.

Admin panel
In your WordPress Admin Panel, Click on Settings > 5,000+ Vector Icons. Or you can also go to your Post / Page Edit. You will see a "Add icon" button just besides "Add Media" button. Click on "Add icon" button to load the 5,000+ Vector Icons panel.
Load Icon Set
First you will see a list of all Icon Sets with the number of icons in each set.
Click on any set to load the icons below.
Click on the loaded icons to select. You can also bookmark (star) them by clicking on the yellow star in the top right corner of every icon. These bookmarked icons will load as starred everytime you load the icon set.


Style your selected Icons
After you have picked your icons, just click on the button below "Click to style your selected icons", to load the CSS3 panel. This panel helps you to add styling options to your icons in bulk.
In the CSS3 Panel, Control the size, padding (surrounding space), margin (outer surronding space), alignments (left, centre or right aligning), background color, background gradient colors, border style, border width, border color, border radius.
Also control the effects. These effects are applied when the user takes the mouse over (on hover) the icons. Take care of the on hover properties in the panel. On hover means the following style is applicable when the user takes mouse over the icons on a website.
After setting your style, Click on "Apply Style" below to see the effect in action below.

Add URL and Insert Shortcode
After applying style to your selected icons, you will see the shortcode generated below (right side of every icon).
You can also add the URL link to every icon in the Add URL text-box. Also set the URL target (ie. open the url link on new web page or same page).
Now if you are viewing this panel on your Text-Editor WordPress Page, you will see a Insert button besides every Icon (Just below the shortcode). Click to insert the shortcode directly in your editor and save the post / page. The icon will be added to your page.
You can also copy - paste this generated shortcode in your Page / Post directly to see the icons.
